# Tetris 俄罗斯方块游戏开发计划

## 项目概述
使用Python语言开发一个经典的俄罗斯方块游戏，支持简体中文界面，画面精美。

## 技术栈
- **语言**: Python 3.10+
- **图形库**: Pygame (用于游戏图形渲染和用户输入处理)
- **打包工具**: PyInstaller (用于打包成可执行文件)

## 操作控制
- **左箭头**: 左移
- **右箭头**: 右移
- **上箭头**: 旋转
- **下箭头**: 软降（加速下落）
- **空格键**: 硬降（直接落到底部）
- **C键**: 保持方块
- **P键**: 暂停/继续
- **R键**: 重新开始
- **M键**: 音效开关
- **ESC键**: 退出游戏
